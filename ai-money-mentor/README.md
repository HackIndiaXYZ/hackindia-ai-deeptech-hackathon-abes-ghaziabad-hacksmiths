AFTERMATH — AI-Powered Financial Shock Therapy

"The best time to invest was 10 years ago. The second best time is NOW." AFTERMATH is a gamified, AI-powered personal finance platform that uses shock, humor, and emotional storytelling to push Indian millennials and Gen-Z to face their financial reality—and improve it.

Built with React, Vite, Tailwind CSS, Framer Motion, GSAP, and Google Gemini AI.

🚀 Quick Start

Prerequisites

Node.js ≥ 18.x npm ≥ 9.x Google Gemini API Key (get it from https://aistudio.google.dev) Installation

Clone repository

git clone https://github.com/Dhruv-9927/AFTERMATH-.git cd AFTERMATH-/ai-money-mentor

Install dependencies

npm install

Create environment file

echo REACT_APP_GEMINI_KEY=your_api_key_here > .env

Run development server

npm run dev

Open: http://localhost:5173
